# Nyabawa Legal Practice

Welcome to the Nyabawa Legal Practice website project! This README provides setup instructions for running the project locally using Vite with React, as well as instructions for deploying it to Vercel.

## Table of Contents

- [Setup Instructions](#setup-instructions)
- [Deployment](#deployment)

## Setup Instructions

Follow these steps to get the project up and running on your local machine:

1. **Clone the Repository**  
   Open your terminal and run:
   ```bash
   git clone https://github.com/cmkwananzi-ui/Nyabawa-Legal-Practice.git
   cd Nyabawa-Legal-Practice
   ```

2. **Install Dependencies**  
   Ensure you have [Node.js](https://nodejs.org/) installed, then run:
   ```bash
   npm install
   ```

3. **Start the Development Server**  
   You can start the local server using Vite by running:
   ```bash
   npm run dev
   ```
   Your application will be available at `http://localhost:3000`.

4. **Open the App**  
   Navigate to `http://localhost:3000` in your browser to view the project.

## Deployment

To deploy your application to Vercel, follow these steps:

1. **Create a Vercel Account**  
   If you don’t already have one, sign up at [vercel.com](https://vercel.com/).

2. **Import Your Project**  
   On your Vercel dashboard, click on `New Project` and import the GitHub repository containing your code.

3. **Configure Vercel Settings**  
   Follow the prompts in Vercel to configure your project settings. Vercel will automatically detect the framework (Vite + React) and set up the build settings for you.

4. **Deploy**  
   Click `Deploy` and Vercel will build and deploy your application. Once completed, you will receive a live link to your deployed site!

Enjoy building your legal practice website!
